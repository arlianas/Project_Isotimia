ΑΝΑΦΟΡΑ ΕΡΓΑΣΙΑΣ PROJECT ΙΣΟΤΙΜΙΑ

 

Στην παρούσα εργασία καλούμαστε να δημιουργήσουμε μια σελίδα η οποία λαμβάνοντας υπόψη τις ισοτιμίες των νομισμάτων, θα μας επιστρέφει το ποσό ενός νομίσματος σε ένα άλλο. Το παραπάνω μπορεί να υλοποιηθεί αρκετά εύκολα αν οι ισοτιμίες των νομισμάτων παρέμεναν σταθερές κατά τη διάρκεια του χρόνου. Όμως οι ισοτιμίες των νομισμάτων αλλάζουν σε καθημερινή βάση και μάλιστα σχεδόν κάθε λεπτό. Είμαστε έτσι υποχρεωμένοι να δημιουργήσουμε μια υλοποίηση η οποία θα λαμβάνει υπόψη αυτή την ιδιαιτερότητα.

Προκειμένου να αναπτύξουμε την παραπάνω υλοποίηση θα χρησιμοποιήσουμε ένα servlet, το οποίο θα καλείται από το html αρχείο που δημιουργούμε και μέσω της βιβλιοθήκης JDOM, θα κάνουμε parsing από ένα αρχείο xml της Ευρωπαϊκής Κεντρικής Τράπεζας, τις ισοτιμίες των νομισμάτων, τις οποίες θα χρησιμοποιήσουμε για να δώσουμε το ποσό που ζητάει ο client στο επιθυμητό νόμισμα. Το αρχείο xml ανανεώνεται καθημερινά, οπότε επιτυγχάνουμε δυναμική υλοποίηση όσον αφορά τις ισοτιμίες.

Αρχικά δημιουργούμε ένα αρχείο το οποίο ονομάζουμε index.jsp. Στη συνέχεια γράφοντας τον κώδικα html και μορφοποιώντας τον με τα κατάλληλα στοιχεία css, δημιουργούμε μια σελίδα στην οποία θα επιλέγουμε τα νομίσματα καθώς και το ποσό του νομίσματος που θέλουμε να μας επιστραφεί βάση της ισοτιμίας τους. Στον κώδικα του αρχείου αυτού δημιουργούμε ένα button το οποίο πατώντας το, καλεί το servlet που θα κάνει την εργασία που θέλουμε. Το αποτέλεσμα εμφανίζεται μετά την κλήση του servlet στην ίδια σελίδα. 

Επίσης στο αρχική μας σελίδα δημιουργήσαμε δύο buttons. Το πρώτο θα καλεί ένα servlet το οποίο θα μας επιστρέφει μια σελίδα με την αναφορά του project και το δεύτερο θα είναι ένα download button από το οποίο θα κατεβάζουμε σε μορφή .rar τα source files του Project μας. Την πρώτη περίπτωση την υλοποιούμε δημιουργώντας έναν listener ο οποίος ουσιαστικά απαρτίζεται από δύο κλάσεις java και ένα servlet το οποίο κάνει redirect σε μια σελίδα στην οποία έχουμε γράψει την αναφορά μας. Τη δεύτερη την υλοποιούμε με ένα servlet το οποίο μας επιστρέφει ένα .rar αρχείο με τις κλάσεις και τα servlets που έχουμε δημιουργήσει στο Project μας, ώστε να είναι διαθέσιμο στον καθένα.

Τέλος για την ολοκλήρωση της εργασίας απαιτήθηκε να ανεβάσουμε τα sources στο network github το οποίο ήταν μια σχετικά πολύπλοκη διαδικασία, καθώς και στον server inf.uth.gr του Πανεπιστημίου Θεσσαλίας προκειμένου το Project μας να είναι προσβάσιμο από μια static ip.

Κατά την υλοποίηση της εργασίας συνάντησα αρκετές δυσκολίες που αφορούσαν την ανάπτυξη του κώδικα των servlets, γι' αυτό χρησιμοποιήσα τη βοήθεια του λογισμικού Netbeans. Επίσης για την υλοποίηση της εργασίας χρησιμοποίησα και τη βοήθεια δύο συμφοιτητριών μου, των Αλεξία Μαρνάρη και Παρασκευή Μικρομάστορα, καθώς επίσης και τη βοήθεια που παρέχονταν από τα σχόλια φοιτητών και βοηθών διδασκαλίας στη σελίδα του μαθήματος.
